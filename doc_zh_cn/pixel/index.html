像素操作

我怎么觉得快速上手章节里，该说的我都说了呢？

既然你点进来了，不能让你空手而归吧

那么送你一小段程序好了：

/* test.c */
#include "tlibmp.h"
#include <stddef.h>

int main(void)
{
    tlb_image_t * image = NULL;
    uint32_t x,y;

    /* Create new image */
    image = tlb_img_new(256,256,tlb_rgb(0,0,0));

    /* Print in this image */
    for(x = 0; x<256; x++){
        for(y = 0; y<256; y++){
            tlb_pixel_set(image, x, y, tlb_rgb(x,0,y));
        }
    }

    /* Save Image */
    tlb_save_bmp(image, "out.bmp");

    /* Free Object */
    tlb_img_free(image);

    return 0;
}
